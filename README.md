# 🐛 GitHub Issues Tracker

A GitHub Issues viewer with login authentication, live API data, and tab-based filtering. Built with vanilla JavaScript and Tailwind CSS.

🔗 **Live Site:** https://roescodynex89-sketch.github.io/hero-ass-05/

## ✨ Features

- 🔐 **Login Page** — Admin credentials required to access the dashboard
- 📋 **3-Tab Filtering** — All, Open, and Closed issue tabs
- 🃏 **Issue Cards** — Title, description, status, author, priority, label, and creation date
- 🟢 **Color-Coded Borders** — Green border for open issues, purple for closed
- 🔍 **Search Bar** — Search issues from the navbar
- 🔄 **Loading Spinner** — Shown while fetching data from the GitHub API
- 💬 **Issue Modal** — Click any card to view full issue details
- 📱 **Fully Responsive** — Works on mobile and desktop

## 📦 Dependencies

This is a vanilla JS project — **no npm packages required.**
Tailwind CSS is loaded via CDN. Data is fetched live from the **GitHub REST API**.
