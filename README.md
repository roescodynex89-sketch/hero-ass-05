
1. What is the difference between var, let, and const?
ANSWER-
 *** var: It can be re-declared and re-assigned.
 ***let: It cannot be re-declared but can be re-assigned. It is block-scoped.
 *** const: It is block-scoped and cannot be re-declared or re-assigned.
2. What is the spread operator (...)?
ANSWER-
  It helps to expand or unpack all elements from an Array or Object separately.

3. What is the difference between map(), filter(), and forEach()?
ANSWER-
 ***map(): It returns a new array.
***filter(): It returns a new array by matching a condition.
 ***forEach(): It is used for looping through an array and does not return anything.

4. What is an arrow function?
ANSWER-
  An arrow function is a short and modern way of writing functions in JavaScript.

5. What are template literals?
ANSWER-
  It is a way to write strings easily using backticks (`) and ${variable}.
